4. 修复说明

修复内容：
1. 在 Features.tsx 的 Image 组件中添加了 style={{ width: 'auto', height: 'auto' }}
2. 这确保了当使用 CSS 类修改图片大小时，宽高比得到正确保持
3. 消除了 Next.js 关于宽高比修改的警告

技术细节：
- 原始设置：width={400}, height={192}, className="w-full h-full"
- 修复后：添加 style={{ width: 'auto', height: 'auto' }}
- 效果：保持图片原始宽高比，避免变形
