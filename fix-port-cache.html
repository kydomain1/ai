🔧 端口缓存问题修复工具

此工具帮助解决浏览器缓存导致的端口不匹配问题。

1. 当前端口状态检测

2. 图片加载测试

3. 缓存清理测试

4. 解决方案步骤

步骤 1: 清理浏览器缓存

Chrome/Edge:

1. 按 Ctrl + Shift + Delete
2. 选择"所有时间"
3. 勾选"缓存的图片和文件"
4. 点击"清除数据"

Firefox:

1. 按 Ctrl + Shift + Delete
2. 选择"全部"
3. 勾选"缓存"
4. 点击"立即清除"

步骤 2: 硬刷新页面

Windows: Ctrl + F5 或 Ctrl + Shift + R
Mac: Cmd + Shift + R

步骤 3: 检查服务器状态

确保服务器运行在端口 3000:
- 访问 http://localhost:3000
- 检查控制台是否有错误
- 确认图片加载正常

步骤 4: 如果问题仍然存在

1. 关闭所有浏览器标签页
2. 重启浏览器
3. 重新访问 http://localhost:3000
4. 检查开发者工具中的网络请求
