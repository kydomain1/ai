🔧 Stripe 版本错误修复测试

此页面用于测试 Stripe API 版本错误的修复。

1. 错误分析

原始错误：

VersionError: Unsupported on version [basil]: Can not provide apiVersion to Stripe()

原因：客户端 Stripe 实例中包含了 apiVersion 参数，但客户端不支持此参数。

2. 修复内容

客户端修复：

// 修复前 return loadStripe(publishableKey, { stripeAccount: undefined, apiVersion: '2024-12-18.acacia', // ❌ 客户端不支持 }); // 修复后 return loadStripe(publishableKey); // ✅ 简化配置

服务器端优化：

// 使用兼容的 API 版本 return new Stripe(stripeSecretKey, { apiVersion: '2024-11-20.acacia', // ✅ 稳定版本 });

6. 版本兼容性说明

Stripe 版本兼容性：

  • 客户端 (@stripe/stripe-js): 不支持 apiVersion 参数
  • 服务器端 (stripe): 支持 apiVersion 参数
  • 当前版本: stripe@18.5.0, @stripe/stripe-js@7.9.0

最佳实践：

1. 客户端：只使用 publishableKey
2. 服务器端：指定稳定的 API 版本
3. 避免在客户端传递服务器端参数
4. 定期更新到稳定版本
